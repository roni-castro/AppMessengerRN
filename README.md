### Getting started

Install dependencies

```
yarn
```

Install iOS pods dependecies

```
cd ios && pod install && cd ..
```

Run iOS project

```
yarn ios
```

Run Android project

```
yarn android
```

### Tests

Run tests

```
yarn test
```

Update snapshots

```
yarn test -u
```
